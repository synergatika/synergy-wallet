# Synergy Wallet

## Contibution Guide

```
mkdir wallet && cd wallet
git clone git@github.com:synergatika/synergy-core.git
git clone git@github.com:synergatika/synergy-wallet.git

cd synergy-core
npm install

cd ../synergy-wallet
npm install

npx ng serve
```
